# 📋 Formulário de Cadastro Moderno

Projeto desenvolvido com foco em **experiência do usuário (UX)**, **validação em tempo real** e **design responsivo**, atendendo a um cenário de problema real de negócio.

---

## 🚀 Sobre o Projeto

Este formulário simula um sistema de cadastro moderno, resolvendo problemas comuns como:

* Falta de feedback visual
* Erros pouco claros
* Falta de responsividade (principalmente em mobile)
* Ausência de validação de senha

A proposta foi criar uma solução simples, porém com **acabamento profissional**.

---

## ✨ Funcionalidades

✔️ Validação em tempo real dos campos
✔️ Feedback visual (cores e mensagens claras)
✔️ Indicador de força da senha
✔️ Confirmação de senha obrigatória
✔️ Modal de sucesso após cadastro
✔️ Layout responsivo (funciona em celular e desktop)
✔️ Design moderno em tema dark

---

## 🧠 Regras de Validação

* **Nome:** mínimo de 3 caracteres
* **Email:** formato válido (regex)
* **Senha:**

  * mínimo de 6 caracteres
  * pelo menos 1 letra maiúscula
  * pelo menos 1 número
* **Confirmação de senha:** deve ser idêntica à senha

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎨 Destaques de UX/UI

* Feedback imediato ao digitar
* Cores semânticas (verde = sucesso, vermelho = erro)
* Animação suave no modal
* Indicador visual de força da senha
* Interface limpa e intuitiva
